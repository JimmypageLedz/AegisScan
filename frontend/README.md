# AegisScan Frontend

这是 AegisScan 的前端界面，负责展示资产、任务、发现、报告以及 LLM 配置等功能。

前端使用 React + TypeScript + Vite 构建，主要用于和后端 API 联调，并提供中文化的可视化操作界面。

## 功能

- 资产管理：新增、查看、删除扫描目标。
- 任务管理：创建扫描任务，查看任务结果，删除任务。
- 发现查看：按任务浏览扫描发现的安全问题。
- 报告生成：调用后端接口生成风险报告。
- 模型设置：配置后端 Base URL、API Key、模式和模型名称。
- 模型列表：可通过后端接口获取可用模型列表后选择。
- 中文界面：所有主要操作和提示都采用中文展示。

## 技术栈

- React
- TypeScript
- Vite
- Fetch API

## 本地启动

```powershell
cd E:\web-security-platform\frontend
npm install
npm run dev
```

启动后访问：

```text
http://localhost:5173
```

## 后端配置

前端需要连接后端 API。默认情况下，请确保后端运行在：

```text
http://127.0.0.1:8000
```

如果后端地址不同，可以在前端设置栏中修改 Base URL。

## 主要页面

- 资产页：管理待扫描的目标。
- 任务页：发起扫描并查看任务状态。
- 发现页：查看任务关联的扫描结果。
- 报告页：生成和查看风险报告。
- 设置页：配置后端地址、API Key 和模型。

## 说明

当前前端主要服务于本地开发和项目展示，重点是把后端接口完整串起来，而不是追求复杂的组件体系。
